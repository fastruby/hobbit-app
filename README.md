# Hobbit App Template

This is a template for projects that want to use the
[Hobbit](https://github.com/patriciomacadden/hobbit) microframework.

## Dependencies

* Hobbit
* Redis
* Ohm
* Puma
* Dotenv
* Rack Protection

Development:

* Minitest
* Shotgun
